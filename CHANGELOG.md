# Changelog

All notable changes to CommandGuard will be documented here.

The project follows semantic versioning once it has a stable release process.

## 0.1.0 - Unreleased

### Added

- Rust CLI with `guard`, `analyze`, and `bench` commands.
- Local analyzer for pasted Bash command effects.
- Bracketed paste scanner for PTY guarded shell mode.
- Bounded base64, hex, and gzip decoding.
- Structured `Analysis` JSON output.
- Inert benchmark corpus for benign, malicious, and mutated commands.
- Beginner-friendly documentation and security model.
