## One-Sentence Pitch

CommandGuard protects Linux users from copied terminal commands that hide, download, or execute risky code before the user understands what will happen.

## Longer Pitch

CommandGuard is a local, explainable defense against ClickFix, pastejacking, and malicious "copy this into your terminal" instructions. It detects bracketed paste, analyzes pasted Bash command effects, decodes simple concealed payloads, and warns before risky text reaches the shell.
